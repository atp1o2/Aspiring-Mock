# Aspire FrontEnd React
Communicates with Jane (Rails API)

## Setup
```
npm install
npm start
```

## Add Heroku Remote Git
```
git remote add heroku https://git.heroku.com/mysterious-crag-56243.git
```

## Deploy to Heroku
```
git pull
npm run build
git push heroku master -f
```
